2.0.0-10 Jan 17, 2017

  • Upgraded all 3rd party JS libraries.
  • Convert to web mercator (#5).
  • Upgrade user management to AGRC Permission Proxy (#4).
  • Switch PLSS layer to vector tiles layer (#1).
  • Enable an anonymous user (#14).
  • Switched to bootstrap for majority of layout components (#16).
  • Upgrade nightly script to be a [forklift](https://github.com/agrc/forklift) pallet.
  • Filter out all data where `REQUEST = 'NO'`
  • Remember the last county on page load.

1.4.0 Sept 3 2013

  • Added all counties to app.

1.3.0 Apr 13 2013

  • Added Kane county to application.

1.2.1 Jul 19 2012

  • Pointed photos layer at single state-wide feature class.
  • Pointed photo urls at roads.utah.gov/wddr.

1.2.0 May 29 2012

  • Added release notes page.
  • Added REQUEST field to Secure role and only showed features where REQUEST = 'YES' for General role.
  • Removed S_WIDTH field from all parts of application.
  • Changed "Affidavits" to read "Exhibit #".
  • Removed "Proposed RR" and "BLM Wilderness Study" layers from General role.
  • Fixed bug where the identify was not honoring the visibility of the roads and photo layers.
  • Fixed bug where the search boxes would not zoom to features that had special characters in them such as "Woods & Eyre Branding Road" in Beaver County.
